# Visual DSA Algorithm

Interactive DSA learning dashboard built with React, Vite, Tailwind CSS, and AI-powered analytics.

## Features

- Dashboard Analytics
- Interview Readiness Tracking
- AI Study Plan
- Company Readiness Insights
- Topic Mastery Tracking
- Activity Heatmap
- Recommended Problems
- AI Coach

## Tech Stack

- React
- Vite
- Tailwind CSS
- Recharts
